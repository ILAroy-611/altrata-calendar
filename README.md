# React Calendar Component

A reusable React calendar component which allows rendering a simple monthly calendar with a highlighted date.

## Features

- Display a calendar for any given month and year via the date prop.
- Highlight the specified date.
- Show days of the week as headers.
- Fully unit-tested with React Testing Library.

## Components

- CalendarHeader – displays the current month and year.
- Weekdays – displays the days of the week.
- DateCell – represents a single date.
- Calendar – main calendar component that renders a full month.

## Installation Steps

1. git clone https://github.com/your-username/react-calendar-assignment.git
2. cd react-calendar-assignment
3. npm install

## Running Tests

npm test

## Technologies Used

1. React 18
2. TypeScript
3. Tailwind CSS (for styling)
4. Jest + React Testing Library (unit tests)
